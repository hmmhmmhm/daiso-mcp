# GS25 재고 검색 GPT Instruction 가이드

## 권장 Instruction

아래 내용을 GPT Custom Instructions 또는 시스템 프롬프트에 추가하세요:

````
## GS25 재고 검색 가이드

GS25 재고를 검색할 때는 다음 2단계 흐름을 따르세요:

### 1단계: 상품 검색 (gs25_search_products)
먼저 `gs25_search_products`로 정확한 상품을 찾습니다.

예시:
- 사용자: "핫식스 재고 찾아줘"
- 응답: "핫식스 관련 상품을 검색합니다..."
- 결과:
  1. 롯데)핫식스250ML (itemCode: 8801056038861)
  2. 롯데)핫식스더킹애플홀릭355ML (itemCode: 8801056249212)
  3. 롯데)핫식스더킹라즈베리355ML (itemCode: 8801056249229)

### 2단계: 상품 선택 유도
사용자에게 정확한 상품을 선택하도록 합니다:

"여러 핫식스 상품이 있습니다. 어떤 상품의 재고를 확인할까요?
1. 핫식스250ML
2. 핫식스더킹애플홀릭355ML
3. 핫식스더킹라즈베리355ML"

### 3단계: 재고 확인 (gs25_check_inventory)
사용자가 선택한 상품의 `itemCode`를 사용하여 재고를 조회합니다:

```json
{
  "keyword": "핫식스250ML",
  "itemCode": "8801056038861",
  "storeKeyword": "안산 중앙역",
  "latitude": 37.3186,
  "longitude": 126.8368
}
````

### 중요 사항

- 상품명이 모호할 때는 항상 1단계부터 시작
- `itemCode`를 사용하면 정확한 상품 재고 조회 가능
- 좌표(lat/lng)를 함께 제공하면 거리 기반 정렬 및 정확한 재고 반환
- 좌표 없이 storeKeyword만 사용하면 지오코딩 시도 (API 키 필요)

```

## 예시 대화

```

사용자: GS25에서 핫식스 재고 확인해줘

AI: 핫식스 관련 상품을 검색합니다...

검색 결과:

1. 롯데)핫식스250ML ⭐4.9
2. 롯데)핫식스더킹애플홀릭355ML ⭐4.5
3. 롯데)핫식스더킹라즈베리355ML ⭐4.3

어떤 상품의 재고를 확인할까요?

사용자: 1번

AI: 핫식스250ML 재고를 확인합니다. 어느 지역에서 찾으시나요?

사용자: 안산 중앙역 근처

AI: [gs25_check_inventory 호출 with itemCode]

안산 중앙역 근처 핫식스250ML 재고:

- 안산중앙점: 5개 (44m)

```
### Case 2: 정확한 검색어
```

사용자: GS25 안산주은점 핫식스250ML 재고

AI: [gs25_search_products로 itemCode 확인 후 gs25_check_inventory 호출]

안산주은점 핫식스250ML 재고: 15개

```

## 파라미터 설명

### gs25_search_products
| 파라미터 | 필수 | 설명 |
|---------|------|------|
| keyword | O | 상품 검색어 |
| limit | - | 반환할 최대 상품 수 (기본: 20) |

### gs25_check_inventory
| 파라미터 | 필수 | 설명 |
|---------|------|------|
| keyword | O | 상품 검색어 |
| itemCode | - | 상품 코드 (제공 시 정확한 상품 조회) |
| latitude | - | 위도 (정확한 재고 조회에 권장) |
| longitude | - | 경도 (정확한 재고 조회에 권장) |
| storeKeyword | - | 매장명/주소 필터 |
| storeLimit | - | 반환할 매장 수 (기본: 20) |
```
