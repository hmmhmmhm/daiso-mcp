# Research Scripts

외부 서비스 구조 분석, 모바일 앱 관찰, 네트워크 캡처 재현 실험용 스크립트를 둡니다.

- GS25 관련 `gs25-*` 스크립트는 앱/네트워크 응답 구조 분석과 재현 실험에 사용합니다.
- `frida/`는 모바일 앱 런타임 관찰용 훅입니다.
- `mitm/`, `mitmproxy/`는 프록시 캡처와 재생 실험용 스크립트입니다.
- 세븐일레븐 관련 `seveneleven-*` 스크립트는 공개/앱 엔드포인트 재현 실험에 사용합니다.

이 경로의 스크립트는 기본 CI 경로에 넣지 않고, 필요한 입력 파일과 산출물을 인접 문서나 파일 상단에 기록합니다.
